#### canvasID

Gets and sets the ID attribute of the canvas HTML tag.

- **Parameters**: {string} paramCanvasID
- **Return Value**: #canvasID

*Example:*

```
gdj1.config.canvasID = 'diagram-1-canvas'; // specify the canvas element's an ID
```
